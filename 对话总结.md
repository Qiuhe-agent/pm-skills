## 三、关键技术概念

### AI产品管理框架
1. **7阶段工作流**：战略规划 → 市场研究 → 产品发现 → 产品执行 → 数据分析 → 市场推广 → 营销增长
2. **PESTEL分析**：政治、经济、社会、技术、环境、法律因素
3. **TAM/SAM/SOM**：市场规模三级评估模型
4. **AARRR漏斗**：获取、激活、留存、收入、推荐指标体系
5. **JTBD（Jobs-to-be-Done）**：用户任务理论

### AI产品特有概念
1. **RAG系统**：检索增强生成，用于知识管理
2. **向量数据库**：存储和检索代码/文档embedding
3. **LLM微调**：针对特定场景优化大模型
4. **护栏指标**：AI产品安全性和质量控制指标
5. **PLG（Product-Led Growth）**：产品主导增长模式

### 运维AI（AIOps）概念
1. **预测性告警**：基于时序数据的异常检测
2. **根因分析**：知识图谱+GNN进行故障定位
3. **智能降噪**：AI聚类减少告警噪音
4. **自动化修复**：Playbook驱动的自动故障处理
5. **MTTR（Mean Time To Repair）**：平均修复时间
