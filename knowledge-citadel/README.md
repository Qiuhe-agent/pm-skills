# 知识城邦 (Knowledge Citadel)

一个游戏化的个人知识管理系统，在 Claude Code 中运行。

## 游戏简介

在数字世界的深处，有一片待开发的土地——知识荒原。每位用户都是这里的开拓者，通过种下知识的种子，建立起属于自己的城邦。知识越丰富，城邦越繁荣。

### 核心隐喻

| 现实概念 | 游戏化映射 |
|---------|-----------|
| 单条笔记/知识点 | 建筑（房屋、工坊、塔楼等）|
| 知识分类 | 城区/区域 |
| 知识关联 | 道路、桥梁 |
| 标签 | 装饰、旗帜 |
| 阅读/复习 | 维护、翻新 |

## 快速开始

```bash
cd knowledge-citadel
python main.py
```

## 游戏命令

| 命令 | 说明 |
|-----|------|
| `/add` | 添加新知识（建造建筑） |
| `/view` | 查看城邦全景 |
| `/list` | 列出所有建筑 |
| `/read <id>` | 阅读/维护建筑 |
| `/link <id1> <id2>` | 建立两个建筑的关联 |
| `/upgrade <id>` | 升级建筑 |
| `/districts` | 查看所有城区 |
| `/add-district <name>` | 添加新城区 |
| `/achievements` | 查看成就 |
| `/status` | 查看详细状态 |
| `/help` | 显示帮助 |
| `/quit` | 退出游戏 |

## 建筑类型

- 📚 **图书馆** - 理论知识、概念定义
- 🏭 **工坊** - 实操技能、步骤方法
- 💡 **灯塔** - 灵感创意、突发想法
- 🏛️ **纪念馆** - 重要记录、里程碑
- 🏪 **仓库** - 资料素材、参考链接

## 资源系统

- 🪙 **智慧币** - 录入知识、日常阅读获得
- ✨ **灵感精华** - 应用知识、建立关联获得
- 📖 **知识卷轴** - 完成成就、里程碑获得
- 👥 **访客数** - 知识分享、输出获得
