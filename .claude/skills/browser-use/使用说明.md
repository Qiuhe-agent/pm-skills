# browser-use 使用说明

`browser-use` 是一个浏览器自动化工具，用于网页测试、表单填写、截图和数据提取。

---

## 快速开始

### 1. 检查安装

```bash
browser-use doctor
```

### 2. 核心工作流程

| 步骤 | 命令 | 说明 |
|------|------|------|
| 1. 打开网页 | `browser-use open <url>` | 导航到指定 URL |
| 2. 查看元素 | `browser-use state` | 获取可点击元素及索引 |
| 3. 交互操作 | `browser-use click 5` | 使用索引进行操作 |
| 4. 验证结果 | `browser-use screenshot` | 确认操作结果 |

---

## 浏览器模式

| 模式 | 命令 | 说明 |
|------|------|------|
| **chromium** (默认) | `browser-use open <url>` | 无头模式，快速隔离 |
| **有头模式** | `browser-use --headed open <url>` | 可见浏览器窗口 |
| **真实 Chrome** | `browser-use --browser real open <url>` | 使用本地 Chrome |
| **带登录状态** | `browser-use --browser real --profile "Default" open <url>` | 使用 Chrome 配置文件 |
| **远程浏览器** | `browser-use --browser remote open <url>` | 云浏览器 |

## 常用命令

### 导航

```bash
browser-use open https://example.com    # 打开网页
browser-use back                          # 返回上一页
browser-use scroll down                   # 向下滚动
browser-use scroll up                     # 向上滚动
browser-use scroll down --amount 1000    # 滚动指定像素
```

### 页面状态

```bash
browser-use state                         # 获取 URL、标题、可点击元素
browser-use screenshot                    # 截图（base64）
browser-use screenshot path.png           # 保存截图到文件
browser-use screenshot --full path.png    # 全屏截图
```

### 交互操作

> **重要**: 先运行 `browser-use state` 获取元素索引！

```bash
browser-use click <index>                 # 点击元素
browser-use type "文本"                    # 在聚焦元素中输入
browser-use input <index> "文本"           # 点击元素后输入
browser-use keys "Enter"                   # 发送按键
browser-use keys "Control+a"               # 发送组合键
browser-use select <index> "选项"          # 选择下拉选项
browser-use hover <index>                  # 悬停
browser-use dblclick <index>               # 双击
browser-use rightclick <index>             # 右键点击
```

### 数据提取

```bash
browser-use eval "document.title"          # 执行 JavaScript
browser-use get title                       # 获取页面标题
browser-use get html                        # 获取完整 HTML
browser-use get html --selector "h1"       # 获取指定元素 HTML
browser-use get text <index>                # 获取元素文本
browser-use get value <index>               # 获取输入框值
browser-use get attributes <index>          # 获取元素属性
```

### 等待条件

```bash
browser-use wait selector "h1"             # 等待元素出现
browser-use wait selector ".loading" --state hidden  # 等待元素消失
browser-use wait text "Success"             # 等待文本出现
browser-use wait selector "h1" --timeout 5000  # 自定义超时（毫秒）
```

### Cookie 管理

```bash
browser-use cookies get                     # 获取所有 cookies
browser-use cookies set <name> <value>      # 设置 cookie
browser-use cookies clear                   # 清除所有 cookies
browser-use cookies export <file>           # 导出 cookies
browser-use cookies import <file>           # 导入 cookies
```

### 会话管理

```bash
browser-use sessions                        # 列出活动会话
browser-use close                           # 关闭当前会话
browser-use close --all                     # 关闭所有会话
```

### Python 执行

```bash
browser-use python "x = 42"                # 设置变量
browser-use python "print(x)"               # 访问变量
browser-use python "print(browser.url)"     # 访问 browser 对象
browser-use python --vars                   # 显示已定义变量
browser-use python --reset                  # 清除 Python 命名空间
browser-use python --file script.py         # 执行 Python 文件
```

## 常见工作流

### 工作流 1: 基础浏览

```bash
# 1. 打开网页
browser-use open https://example.com

# 2. 查看可用元素
browser-use state

# 3. 点击某个元素（索引从 state 获取）
browser-use click 3

# 4. 输入文本
browser-use input 5 "搜索关键词"

# 5. 截图确认
browser-use screenshot result.png

# 6. 完成后关闭
browser-use close
```

### 工作流 2: 使用本地 Chrome 配置文件（带登录状态）

```bash
# 1. 查看可用的 Chrome 配置文件
browser-use -b real profile list

# 2. 使用配置文件打开已登录的网站
browser-use --browser real --profile "Default" open https://github.com

# 3. 操作...
browser-use state
browser-use click 5

# 4. 关闭
browser-use close
```

### 工作流 3: 隧道访问本地开发服务器

```bash
# 1. 启动本地开发服务器（假设在 3000 端口）
# npm run dev

# 2. 创建隧道
browser-use tunnel 3000
# 返回: https://abc.trycloudflare.com

# 3. 使用云浏览器访问
browser-use --browser remote open https://abc.trycloudflare.com

# 4. 操作...
browser-use state
browser-use screenshot

# 5. 清理
browser-use close
browser-use tunnel stop 3000
```

## 提示与技巧

1. **总是先运行 `browser-use state`** - 查看可用元素及其索引
2. **使用 `--headed` 调试** - 可以看到浏览器在做什么
3. **会话会持续** - 浏览器在命令之间保持打开状态
4. **使用 `--json`** - 便于程序化解析
5. **Python 变量持久化** - 同一会话中 `browser-use python` 命令间变量保留
6. **命令别名**: `bu`、`browser`、`browseruse` 都等同于 `browser-use`

## 故障排除

### 浏览器无法启动？

```bash
browser-use close --all           # 关闭所有会话
browser-use --headed open <url>   # 尝试可见窗口模式
```

### 找不到元素？

```bash
browser-use state                 # 检查当前元素
browser-use scroll down           # 元素可能在下方
browser-use state                 # 再次检查
```

### 会话问题？

```bash
browser-use sessions              # 检查活动会话
browser-use close --all           # 清理全部
browser-use open <url>            # 重新开始
```

---

## 清理

完成后始终关闭浏览器：

```bash
browser-use close                     # 关闭浏览器会话
browser-use session stop --all        # 停止云会话（如有）
browser-use tunnel stop --all         # 停止隧道（如有）
```
