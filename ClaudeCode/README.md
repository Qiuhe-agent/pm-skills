## 产品概要

### Claude Code 愿景
"让每一位开发者都拥有一位 24/7 的智能编程伙伴，释放人类的创造潜能。"

### 核心差异化
- **深度代码库理解**：理解整个项目，不只是单文件
- **多步骤工作流自动化**：执行完整任务，不只是单步操作
- **可扩展的技能系统**：用户可以自定义和共享技能
- **跨平台统一体验**：终端、VS Code、JetBrains 等

### 目标市场
- 初期：专业开发者（2-5 年经验）
- 中期：小团队和初创公司
- 长期：企业工程部门

### 市场规模
- TAM：$150 亿/年
- SAM：$40 亿/年
- SOM（3 年目标）：$10.5 亿/年
