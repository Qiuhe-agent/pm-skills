# 知识城邦游戏指南

## 游戏概述

知识城邦是一个游戏化的个人知识管理系统。用户通过输入、整理和关联自己的知识，来建设和发展一个虚拟的"知识城邦"。

## 核心玩法

### 1. 建造建筑（添加知识）

每条知识都是城邦中的一座建筑。根据知识的内容选择合适的建筑类型：

| 建筑类型 | Emoji | 适用场景 |
|---------|-------|---------|
| 图书馆 | 📚 | 理论知识、概念定义、原理说明 |
| 工坊 | 🏭 | 实操技能、步骤方法、教程指南 |
| 灯塔 | 💡 | 灵感创意、突发想法、未来设想 |
| 纪念馆 | 🏛️ | 重要记录、里程碑、难忘时刻 |
| 仓库 | 🏪 | 资料素材、参考链接、收藏内容 |

### 2. 规划城区（知识分类）

城邦分为不同区域，用于组织不同类型的知识：

- 学习区 - 学习相关的知识
- 工作区 - 工作相关的技能
- 生活区 - 生活中的点滴
- 创意区 - 灵感与创意

可以根据需要添加新的城区。

### 3. 修建道路（建立关联）

在知识点之间建立关联，就像在建筑之间修建道路。关联越多，城邦的"知识流动"效率越高。

### 4. 升级建筑

使用灵感精华升级建筑，提升建筑的等级和产出。

## 资源系统

| 资源 | Emoji | 获取方式 | 用途 |
|-----|-------|---------|------|
| 智慧币 | 🪙 | 录入知识、日常阅读 | 建造基础建筑 |
| 灵感精华 | ✨ | 建立关联、应用知识 | 升级建筑 |
| 知识卷轴 | 📖 | 完成成就、里程碑 | 解锁特殊建筑 |
| 访客数 | 👥 | 知识分享、输出 | 提升城邦声望 |

## 成就系统

### 初级成就
- 🌱 初来乍到 - 添加第一条知识
- 🏘️ 小有规模 - 添加10条知识

### 中级成就
- 🛤️ 四通八达 - 建立10条关联
- 🎯 区域专精 - 某个分类满20条

### 高级成就
- 🏰 知识帝国 - 添加100条知识
- 🌆 智慧之城 - 解锁5个区域

### 传奇成就
- 📚 活的百科 - 添加500条知识
- 🕸️ 知识互联 - 建立100条关联

## 命令参考

### scripts/citadel.py

```bash
# 加载或初始化城邦
python scripts/citadel.py load

# 添加知识
python scripts/citadel.py add <标题> <内容> [--type TYPE] [--district DISTRICT] [--tags TAGS]

# 查看城邦
python scripts/citadel.py view

# 列出所有建筑
python scripts/citadel.py list

# 阅读建筑
python scripts/citadel.py read <ID>

# 建立关联
python scripts/citadel.py link <ID1> <ID2>

# 查看成就
python scripts/citadel.py achievements
```

### 建筑类型参数
- `library` - 图书馆
- `workshop` - 工坊
- `lighthouse` - 灯塔
- `memorial` - 纪念馆
- `warehouse` - 仓库

## 数据格式

城邦数据存储在 `knowledge-citadel.md` 文件中，采用 Markdown 格式，包含：

1. YAML frontmatter - 元数据和资源
2. 城区列表
3. 建筑列表（每个建筑是一个二级标题，带表格和内容）
4. 成就记录

这种格式既人类可读，又方便解析。
