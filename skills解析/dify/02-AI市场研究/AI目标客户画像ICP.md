# Dify AI目标客户画像（ICP）

## 概述

理想客户画像（Ideal Customer Profile, ICP）定义了Dify最有可能成功使用产品并从中获得价值的客户类型。

---

## 一、ICP定义

### 1.1 核心ICP陈述

> **Dify的理想客户是：拥有AI应用开发需求、重视数据安全和自主可控、希望通过低代码方式快速构建AI应用的中大型企业技术团队。**

### 2.1 企业规模

| 维度 | 理想范围 | 说明 |

| **年收入** | $10M-$500M | 有支付能力 |
| **技术团队** | 10-100人 | 有开发能力 |

| 优先级 | 行业 | 匹配度 | 典型场景 |
|--------|------|--------|----------|
| ⭐⭐⭐⭐⭐ | 金融科技 | 高 | 智能客服、风控、投顾 |
| ⭐⭐⭐⭐⭐ | 企业服务 | 高 | 内部工具、知识管理 |
| ⭐⭐⭐⭐⭐ | 互联网 | 高 | 产品智能化、运营工具 |
| ⭐⭐⭐⭐ | 教育 | 中高 | 智能辅导、内容生成 |
| ⭐⭐⭐⭐ | 医疗健康 | 中高 | 辅助诊断、病历分析 |
| ⭐⭐⭐ | 制造业 | 中 | 质检、预测性维护 |
| ⭐⭐⭐ | 零售电商 | 中 | 客服、推荐系统 |

### 2.3 地域分布

| 优先级 | 地域 | 特点 |
|--------|------|------|

| ⭐⭐⭐⭐ | 中国二线城市 | 成本优势，需求增长快 |

## 三、技术特征（Technographic）

### 3.1 技术栈要求

| 维度 | 要求 | 说明 |
|------|------|------|
| **云基础设施** | AWS/Azure/阿里云/腾讯云 | 支持主流云平台 |
| **容器化** | Docker/Kubernetes | 便于部署和扩展 |
| **编程语言** | Python/JavaScript/Go | 主流开发语言 |
| **数据库** | PostgreSQL/MongoDB | 支持主流数据库 |

### 3.2 AI能力现状

| 维度 | 理想状态 | 说明 |
|------|----------|------|
| **AI使用经验** | 有一定AI应用经验 | 了解AI能力和局限 |
| **LLM使用** | 已使用或计划使用LLM | 有明确的LLM应用需求 |
| **数据资产** | 有结构化/非结构化数据 | 有知识库建设需求 |
| **AI团队** | 有1-5人AI相关岗位 | 有基本AI能力 |

### 3.3 技术决策特征

| 特征 | 描述 |
|------|------|
| **开源偏好** | 倾向于使用开源技术，重视自主可控 |
| **私有化需求** | 有数据安全要求，需要私有化部署 |
| **技术自主** | 希望掌握核心技术，减少供应商依赖 |
| **成本敏感** | 关注TCO，追求性价比 |

## 五、行为特征（Behavioral）

### 5.1 购买行为

| 特征 | 描述 |
|------|------|
| **决策周期** | 1-3个月（中小企业）/ 3-6个月（大型企业） |
| **决策者** | CTO/技术总监/研发负责人 |
| **影响者** | 架构师、技术负责人、产品经理 |
| **评估方式** | POC测试、技术评审、商务谈判 |
| **购买渠道** | 官网、销售团队、合作伙伴 |

### 5.2 使用行为

| 特征 | 描述 |
|------|------|
| **试用行为** | 先本地部署测试，再决定采购 |
| **学习方式** | 文档自学 + 社区支持 + 培训服务 |
| **使用频率** | 日常高频使用 |
| **扩展需求** | 从单一场景扩展到多场景 |

### 5.3 信息获取渠道

| 渠道 | 重要性 | 说明 |

| **同行推荐** | ⭐⭐⭐⭐ | 口碑传播 |
| **行业会议** | ⭐⭐⭐ | 技术大会、行业展会 |

| **企业规模** | 20% | 100-5000人=5分，其他=3分 |

| **预算匹配** | 10% | 预算充足=5分，一般=3分，不足=1分 |

| 特征 | 成功客户占比 |

## 八、ICP应用指南

### 8.1 销售线索评分

| 总分 | 等级 | 处理策略 |
|------|------|----------|
| 20-25分 | A级（高价值） | 优先跟进，销售重点投入 |
| 15-19分 | B级（中价值） | 正常跟进，培育转化 |
| 10-14分 | C级（低价值） | 自动化培育，等待时机 |
| <10分 | D级（不匹配） | 暂不投入资源 |

### 8.2 营销内容定制

| ICP类型 | 内容重点 | 渠道 |
|---------|----------|------|
| 中型科技公司 | 快速上线、成本优势 | 技术社区、GitHub |
| 大型传统企业 | 安全合规、私有化 | 行业会议、直销 |
| AI服务商 | 效率提升、白标能力 | 合作伙伴渠道 |
