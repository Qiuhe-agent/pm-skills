# AI产品经理工作流用户旅程

## 概述

本文档展示了AI产品经理在完整工作流程中如何使用65个技能。AI产品经理的工作流程与传统PM有所不同，更强调数据驱动、快速实验、AI模型监控和用户反馈分析。

---

## 阶段一：AI战略规划

### 🎯 目标
定义AI产品愿景、AI策略和商业模式，为AI产品开发奠定基础。

### � 产出物
- AI产品愿景声明
- AI产品战略文档
- AI商业模式设计文档（SaaS/API/数据服务）
- AI价值主张陈述
- AI风险评估报告（技术风险、伦理风险、监管风险）

### � 使用技能

| 技能编号 | 技能名称 | AI产品经理应用场景 | 核心输出 |
|---------|----------|------------------|----------|
| **58** | 产品愿景 | 定义AI产品的长期愿景和方向 | AI产品愿景、团队对齐 |
| **57** | 产品策略 | 制定AI产品策略和优先级 | AI战略规划、路线图制定 |
| **51** | 商业模式 | AI产品商业模式分析 | AI SaaS/API/数据服务模式设计 |
| **52** | 精益画布 | AI产品精益画布 | AI商业模式可视化、验证 |
| **59** | 创业画布 | AI创业公司画布 | AI创业规划、投资者演示 |
| **50** | 安索夫矩阵 | AI产品增长策略分析 | AI功能扩展、市场扩张 |
| **54** | PESTLE分析 | AI行业宏观环境分析 | AI监管、技术趋势、伦理考量 |
| **55** | 波特五力 | AI行业竞争力分析 | AI竞争格局、技术壁垒 |
| **61** | 价值主张 | AI产品价值主张创建 | AI价值定位、差异化优势 |

### 🔄 AI产品经理工作流程

```
1. 产品愿景 (58)
   │
   ├─→ 定义AI产品的长期愿景
   │
   ├─→ 产品策略 (57)
   │   │
   │   ├─→ 确定AI产品战略方向
   │   │
   │   ├─→ 商业模式 (51)
   │   │   │
   │   │   └─→ 分析AI商业模式（SaaS/API/数据服务）
   │   │
   │   ├─→ 精益画布 (52) 或 创业画布 (59)
   │   │   │
   │   │   └─→ 可视化AI商业模式
   │   │
   │   ├─→ 安索夫矩阵 (50)
   │   │   │
   │   │   └─→ 选择AI产品增长路径
   │   │
   │   ├─→ PESTLE分析 (54)
   │   │   │
   │   │   └─→ 评估AI监管、技术趋势、伦理因素
   │   │
   │   ├─→ 波特五力 (55)
   │   │   │
   │   │   └─→ 分析AI竞争格局和技术壁垒
   │   │
   │   └─→ 价值主张 (61)
   │       │
   │       └─→ 定义AI产品价值定位和差异化
   │
   └─→ 进入AI市场研究阶段
```

### 💡 AI产品经理关键决策点
- AI产品愿景是否清晰？
- AI商业模式是否可持续？
- 是否考虑了AI监管和伦理问题？
- AI价值主张是否具有差异化？

## 阶段二：AI市场研究

### 🎯 目标
深入了解AI市场、竞争对手和目标用户，为AI产品决策提供数据支持。

### 📦 产出物
- AI市场规模评估报告
- AI竞争对手分析报告
- AI目标客户画像（ICP）
- AI用户画像和细分报告
- AI客户旅程地图
- AI用户情感分析报告

### 📋 使用技能

| 技能编号 | 技能名称 | AI产品经理应用场景 | 核心输出 |
|---------|----------|------------------|----------|
| **28** | 市场规模估算 | AI市场规模和增长预测 | AI市场机会评估、投资决策 |
| **25** | 竞争对手分析 | AI竞争对手优劣势分析 | AI产品竞争分析、技术对比 |
| **27** | 市场细分 | AI市场细分群体 | AI应用场景细分、目标市场定义 |
| **24** | 理想客户画像 | AI产品理想客户画像 | B2B/B2C客户定义 |
| **30** | 用户画像 | AI产品用户画像 | AI用户特征、使用习惯 |
| **31** | 用户细分 | AI用户群体细分 | AI用户行为分析、个性化策略 |
| **26** | 客户旅程地图 | AI产品客户完整旅程 | AI用户体验设计、AI交互流程 |
| **29** | 情感分析 | AI产品用户反馈情感分析 | AI满意度、信任度、偏见分析 |

### 🔄 AI产品经理工作流程

```
AI战略规划阶段
   │
   └─→ 市场规模估算 (28)
       │
       ├─→ 评估AI市场规模和增长趋势
       │
       ├─→ 市场细分 (27)
       │   │
       │   └─→ 划分AI应用场景和细分市场
       │
       ├─→ 理想客户画像 (24)
       │   │
       │   ├─→ 定义AI产品目标客户（B2B/B2C）
       │   │
       │   └─→ 用户画像 (30)
       │       │
       │       └─→ 创建AI用户详细画像
       │
       ├─→ 用户细分 (31)
       │   │
       │   └─→ 细分AI用户群体（技术熟练度、行业等）
       │
       ├─→ 竞争对手分析 (25)
       │   │
       │   ├─→ 识别AI竞争对手
       │   │
       │   ├─→ 分析AI技术能力对比
       │   │
       │   └─→ 评估AI产品差异化优势
       │
       ├─→ 客户旅程地图 (26)
       │   │
       │   └─→ 映射AI产品完整用户旅程
       │
       └─→ 情感分析 (29)
           │
           └─→ 分析AI用户反馈和情感（信任度、满意度、偏见）
           │
           └─→ 进入AI产品发现阶段
```

### 💡 AI产品经理关键决策点
- AI市场规模是否足够大？
- 目标AI用户是否清晰？
- AI竞争对手有哪些技术优势？
- AI用户对AI的信任度如何？
