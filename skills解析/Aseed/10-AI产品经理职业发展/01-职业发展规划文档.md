# Aseed AI产品经理职业发展规划

**产品：** Aseed
**版本：** v1.0
**日期：** 2026年3月

---

## Aseed PM能力模型

### 层级1：AI产品经理（执行层）
**专注：** 把事做对

**核心能力：**
- [ ] 理解AI基本概念（ML、LLM、知识图谱等）
- [ ] 能与AI/ML团队有效沟通
- [ ] 能设计AI产品交互（置信度、反馈、人工兜底）
- [ ] 能定义AI效果指标（准确率、召回率等）
- [ ] 能收集和标注AI训练数据
- [ ] 能编写AI相关的用户故事和验收标准

**成长时间：** 0-1年

---

### 层级2：AI产品负责人（策略层）
**专注：** 做对的事

**核心能力：**
- [ ] 理解AI技术边界和局限性
- [ ] 能制定AI产品路线图
- [ ] 能权衡AI风险和收益
- [ ] 能建立AI产品的反馈闭环
- [ ] 能管理AI技术债务
- [ ] 能跨团队协作（AI、工程、客户成功）
- [ ] 理解AI伦理和治理

**成长时间：** 1-3年

---

### 层级3：AI产品总监（管理层）
**专注：** 搭建体系

**核心能力：**
- [ ] 能制定AI产品战略
- [ ] 能建立AI产品团队
- [ ] 能建立AI产品的流程和规范
- [ ] 能管理AI产品组合
- [ ] 能建立AI治理体系
- [ ] 能向高管沟通AI产品价值
- [ ] 能建立AI生态

**成长时间：** 3-5年
