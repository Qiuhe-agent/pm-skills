## 二、详细客户旅程

### 阶段1：问题认知

**用户状态**：
- 意识到有痛点，但不知道Coze
- 可能在寻找解决方案，但方向不明确

**关键场景**：
1. **刷到内容**：在B站、小红书看到别人用Coze做的Bot
2. **同行交流**：和同行聊天，听说有个叫Coze的工具很好用
3. **遇到痛点**：客服忙不过来、粉丝互动不过来、想做Bot但不会写代码
4. **看到广告**：在百度、抖音刷到Coze的广告

**用户心理**：
```
┌─────────────────────────────────────────────────────────────┐
│              问题认知阶段的用户心理                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  "客服成本太高了，要是有个自动回复的工具就好了"              │
│                      ↓                                       │
│  "听说最近有个叫Coze的工具，好像可以做Bot"                   │
│                      ↓                                       │
│  "真的有这么好吗？我先了解一下"                               │
│                      ↓                                       │
│  （好奇心被勾起，开始搜索）                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**关键触点**：
- 内容平台（B站、小红书、抖音）
- 搜索引擎（百度、谷歌）
- 社交推荐（朋友圈、社群）
- 广告投放

**痛点**：
- 不知道有这个工具
- 不相信真的能解决问题
- 担心学习成本高

**机会**：
- 内容种草，展示成功案例
- KOL推荐，建立信任
- 痛点营销，激发需求

**Coze动作**：
- 产出优质教程和案例内容
- 与KOL、MCN合作
- SEO/SEM优化，让用户能搜到
- 广告投放，精准触达
