## 阶段说明

### 01-AI战略规划

**目标**：定义Coze的愿景、使命、战略目标和商业模式。

**产出物**：
- AI产品愿景声明 - Coze的愿景和长期目标
- AI产品战略文档 - 产品战略、竞争策略、执行计划
- AI价值主张陈述 - 分客户群的价值主张
- AI商业模式设计 - 商业模式画布、定价策略、财务预测
- AI风险评估报告 - 技术、市场、政策风险分析

---

### 02-AI市场研究

**目标**：分析市场规模、竞争对手、目标用户、客户旅程。

**产出物**：
- AI市场规模评估 - TAM-SAM-SOM分析、区域市场、行业应用
- AI竞争对手分析 - 主要竞品优劣势、竞争策略
- AI目标客户画像 - 分客户群的详细画像
- AI用户画像和细分 - 用户细分、需求动机、生命周期
- AI客户旅程地图 - 7阶段客户旅程、触点、痛点
- AI用户情感分析 - 正面/负面情感、用户期待

---

### 03-AI产品发现

**目标**：收集功能创意、排序优先级、验证假设、设计实验。

**产出物**：
- AI功能创意清单 - 50+功能创意，按优先级分类
- AI功能优先级排序 - RICE评分模型、季度路线图
- AI机会解决方案树 - OST框架、5个预期结果的解决方案
- AI假设验证计划 - 6个核心假设、验证方法、时间线
- AI用户访谈脚本 - 完整访谈提纲、洞察模板
- AI实验设计文档 - A/B测试、MVP测试、样本量计算

## 使用方法

### 产品经理工作流

1. **战略规划** - 从01-AI战略规划开始，明确愿景和战略
2. **市场研究** - 用02-AI市场研究的产出物理解市场和用户
3. **产品发现** - 通过03-AI产品发现定义功能和验证假设
4. **产品执行** - 进入04阶段进行开发和发布
5. **数据与增长** - 通过05-07阶段进行数据分析和增长

- positioning-statement - 定位陈述

- jobs-to-be-done - JTBD分析

- user-story - 用户故事
- roadmap-planning - 路线图规划

## Coze 核心要点

### 产品定位
- 多端零代码AI Bot开发平台
- 让AI Bot开发触手可及

### 核心优势
1. 多端发布 - 一次开发，20+平台一键发布
2. 零代码 - 可视化编排，小白也能上手
3. 插件生态 - 1000+现成插件，快速扩展
4. Bot商店 - 创作-发布-变现闭环
5. 中国本地化 - 微信、抖音、飞书深度整合

### 目标用户
1. 内容创作者/社群运营 - UP主、博主、群主
2. 中小企业运营者 - 电商、教育、本地生活
3. 独立开发者/创业者 - MVP验证、效率工具
4. 企业数字化团队 - 内部工具、Bot中台

### 商业模式
- 免费增值（Freemium）
- 个人版 ¥29/月，团队版 ¥199/人/月
- 企业版 ¥99,999/年起
- Bot商店和插件市场分成（平台30%，创作者70%）
