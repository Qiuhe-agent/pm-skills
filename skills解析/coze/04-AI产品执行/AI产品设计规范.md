# Coze AI产品设计规范

## 设计系统概述

本文档定义Coze产品的设计原则、视觉规范、组件规范、交互规范，确保产品体验的一致性。

---

## 1. 设计原则

### 1.1 核心设计原则

**简单直观**
- 降低认知负担，让用户一看就懂
- 优先展示核心功能，隐藏高级功能
- 提供清晰的视觉引导

**高效实用**
- 聚焦用户目标，减少不必要的步骤
- 提供快捷操作，提升效率
- 智能预判用户需求

**友好包容**
- 尊重不同技术背景的用户
- 提供新手引导和帮助
- 及时反馈，建立信任

**现代美感**
- 简洁现代的视觉风格
- 一致的视觉语言
- 细节处体现品质


## 4. 交互规范

### 4.1 新手引导

**引导原则**
- 简短：不超过5步
- 聚焦：只引导核心功能
- 可跳过：随时可以退出

**引导类型**
1. **浮层引导** - 高亮元素+气泡提示
2. **交互式任务** - 引导用户完成具体操作
3. **视频教程** - 可选的视频学习
4. **模板推荐** - 推荐合适的模板

### 4.2 反馈机制

**即时反馈**
- 按钮点击：视觉反馈（变色、缩放）
- 表单提交：加载状态
- 操作成功：成功提示（Toast）
- 操作失败：错误提示

**Toast提示**
```
位置：顶部居中
背景：#323232（深色）
文字：白色
圆角：8px
内边距：12px 24px
显示时长：2-3秒
自动消失
```

**错误处理**
- 内联错误：表单字段下方显示错误
- 弹窗错误：严重错误用弹窗
- 帮助链接：提供"查看帮助"链接

### 4.3 拖拽交互

**画布拖拽**
- 节点拖拽：跟随鼠标移动
- 连线拖拽：起点→终点
- 框选拖拽：选中多个节点
- 画布平移：拖拽空白处或滚轮
- 缩放：Ctrl+滚轮或缩放按钮

**拖拽视觉反馈**
- 拖拽中：半透明+阴影
- 可放置：绿色高亮
- 不可放置：红色禁止
- 吸附效果：智能对齐辅助线

### 4.4 键盘快捷键

**常用快捷键**
```
Ctrl/Cmd + S：保存
Ctrl/Cmd + Z：撤销
Ctrl/Cmd + Shift + Z：重做
Ctrl/Cmd + C：复制
Ctrl/Cmd + V：粘贴
Ctrl/Cmd + D：删除
Ctrl/Cmd + /：搜索
Esc：关闭弹窗/取消
```

## 6. 响应式设计

### 6.1 断点

```
移动端：< 768px
平板端：768px - 1024px
桌面端：> 1024px
```

### 6.2 移动端适配

**导航**
- 顶部导航简化
- 侧边栏改为底部Tab或汉堡菜单

**画布**
- 简化操作
- 支持触摸手势
- 优化节点大小和间距

## 7. 无障碍设计

### 7.1 可访问性

- 键盘导航完整支持
- 屏幕阅读器支持（ARIA标签）
- 颜色对比度符合WCAG AA标准
- 文字缩放支持

### 7.2 包容性设计

- 支持深色模式
- 支持多语言
- 提供多种字体大小选项
- 考虑不同认知能力的用户
