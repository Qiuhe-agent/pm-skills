## 2. 北极星指标

### 2.1 月活创作者数（MAC）

**定义**：月内至少创建或编辑过1个Bot的用户数

**计算公式**：
```
MAC = COUNT(DISTINCT user_id)
WHERE action IN ('bot_create', 'bot_edit', 'bot_publish')
  AND time >= month_start
  AND time < month_end
```

**目标值**：
- Q1 2026：10,000
- Q2 2026：50,000
- Q3 2026：100,000
- Q4 2026：150,000

**为什么选这个指标**：
1. **反映核心价值** - 用户真的在使用产品创作
2. **可测量** - 数据清晰，容易统计
3. **可行动** - 可以通过产品优化来提升
4. **先导性** - 先有创作者，才有收入

## 3. AARRR指标

### 3.1 获取（Acquisition）

| 指标 | 定义 | 目标 |
|------|------|------|
| **曝光量** | 看到产品的人数 | 1000万/月 |
| **点击量** | 点击进入的次数 | 50万/月 |
| **点击率（CTR）** | 点击量 / 曝光量 | 5%+ |
| **访问数** | 网站/App访问次数 | 20万/月 |
| **独立访客（UV）** | 独立访客数 | 15万/月 |
| **注册数** | 完成注册的用户数 | 5万/月 |
| **注册转化率** | 注册数 / 访问数 | 25%+ |

**渠道细分**：
| 渠道 | 占比目标 | CAC目标 |
|------|---------|---------|
| 自然搜索 | 30% | ¥50 |
| 内容营销 | 25% | ¥80 |
| 口碑推荐 | 20% | ¥10 |
| 付费广告 | 15% | ¥200 |
| 其他 | 10% | ¥100 |

### 3.2 激活（Activation）

| 指标 | 定义 | 目标 |
|------|------|------|
| **首日激活率** | 注册24小时内创建Bot的比例 | 30%+ |
| **7天激活率** | 注册7天内创建Bot的比例 | 50%+ |
| **新手引导完成率** | 完成新手引导的比例 | 70%+ |
| **首个Bot创建时间** | 从注册到首个Bot的平均时间 | <30分钟 |
| **首个Bot发布率** | 创建Bot后发布的比例 | 80%+ |

**激活漏斗**：
```
注册用户
    ↓ 100%
完成新手引导
    ↓ 70%（目标）
创建首个Bot
    ↓ 71%（目标）
发布Bot
    ↓ 80%（目标）
激活成功（50%）
```

### 3.3 留存（Retention）

| 指标 | 定义 | 目标 |
|------|------|------|
| **次日留存** | Day1仍活跃的比例 | 50%+ |
| **7日留存** | Day7仍活跃的比例 | 35%+ |
| **30日留存** | Day30仍活跃的比例 | 25%+ |
| **月留存** | 本月活跃用户下月仍活跃 | 40%+ |
| **DAU/MAU** | 日活 / 月活 | 20%+ |

**留存曲线**：
```
100% ┤●
 80% ┤ ●
 60% ┤   ●
 40% ┤     ●●●●●●●
 20% ┤
  0% └─────────────────
     Day1 Day7 Day30
```

**分群留存**：
- 按用户类型：个人、企业、开发者
- 按首次使用场景：模板、空白、导入
- 按活跃度：高频、中频、低频

### 3.4 变现（Revenue）

| 指标 | 定义 | 目标 |
|------|------|------|
| **免费→付费转化率** | 免费用户升级付费的比例 | 8%+ |
| **付费用户数** | 付费用户总数 | 10,000+（Q4） |
| **MRR** | 月度经常性收入 | $40万+（Q4） |
| **ARR** | 年度经常性收入 | $500万+（Q4） |
| **ARPU** | 每用户平均收入 | ¥100/月 |
| **ARPPU** | 每付费用户平均收入 | ¥300/月 |
| **LTV** | 用户生命周期价值 | ¥2,000+ |
| **CAC** | 用户获取成本 | ¥150 |
| **LTV:CAC** | 生命周期价值/获取成本 | 3:1+ |
| **净收入留存（NRR）** | 收入留存率 | 120%+ |
| **退款率** | 退款用户比例 | <5% |

**收入构成**：
| 收入来源 | 占比 |
|---------|------|
| 个人版订阅 | 25% |
| 团队版订阅 | 35% |
| 企业版授权 | 25% |
| 交易分成 | 10% |
| 增值服务 | 5% |

### 3.5 推荐（Referral）

| 指标 | 定义 | 目标 |
|------|------|------|
| **NPS** | 净推荐值 | 40+ |
| **推荐率** | 推荐过产品的用户比例 | 20%+ |
| **病毒系数（k-factor）** | 每个用户带来的新用户 | 0.3+ |
| **推荐带来的新用户** | 推荐渠道的新用户占比 | 20%+ |
| **分享率** | 分享内容/作品的用户比例 | 15%+ |

**NPS计算**：
```
NPS = %推荐者 - %贬损者

推荐者（Promoters）：评分9-10分
被动者（Passives）：评分7-8分
贬损者（Detractors）：评分0-6分
```

## 7. 分析框架

### 7.1 分群分析

**用户分群维度**：
- 用户类型：个人、中小企业、企业、开发者
- 活跃度：高频（每周7次+）、中频、低频
- 生命周期：新用户、成长用户、成熟用户、流失用户
- 渠道来源：自然搜索、内容、付费、推荐
- 地区：国内、海外、一线城市、其他
- 设备：Web、移动端、平板

### 7.2 漏斗分析

**核心漏斗**：
- 获客漏斗：曝光 → 点击 → 访问 → 注册
- 激活漏斗：注册 → 新手引导 → Bot创建 → 发布
- 付费漏斗：免费 → 试用 → 付费 → 续购
- 生态漏斗：浏览 → 安装 → 使用 → 创建 → 分享

### 7.3 留存分析

**留存曲线**：
- 次日、7日、30日留存
- 分群留存对比
- 同期群（Cohort）分析
- 留存影响因素分析

**流失分析**：
- 流失用户画像
- 流失节点分析
- 流失原因调研
- 流失召回效果

   ↓
