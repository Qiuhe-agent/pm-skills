# Flash AI运维产品 推荐画布

## 推荐画布概述

**产品名称**：Flash - AI驱动的智能运维平台
**画布用途**：记录AI驱动的产品推荐策略和规则
**更新日期**：2026年3月

---

## 1. 推荐场景定义

### 1.1 核心推荐场景

| 场景ID | 场景名称 | 触发条件 | 推荐目标 |
|--------|---------|---------|---------|
| RS-01 | 告警关联推荐 | 新告警产生 | 推荐相关历史告警和案例 |
| RS-02 | 根因推荐 | 告警聚合完成 | 推荐可能的根因 |
| RS-03 | 修复推荐 | 根因确定 | 推荐修复方案 |
| RS-04 | 知识推荐 | 用户查询 | 推荐相关知识 |
| RS-05 | 功能推荐 | 用户行为分析 | 推荐未使用的功能 |

### 1.2 推荐场景详情

**RS-01：告警关联推荐**

```
触发条件：
- 新告警进入系统
- 告警完成初步处理

推荐内容：
- 历史相似告警（相似度>70%）
- 关联的系统组件告警
- 同一故障链的告警

推荐位置：
- 告警详情页侧边栏
- 告警列表页关联标签
```

**RS-02：根因推荐**

```
触发条件：
- 告警聚合完成
- 用户请求AI分析

推荐内容：
- AI分析的根因（置信度排序）
- 证据链数据
- 历史类似案例的根因

推荐位置：
- 告警详情页AI诊断区
- 根因分析报告
```

**RS-03：修复推荐**

```
触发条件：
- 根因分析完成
- 用户请求修复建议

推荐内容：
- 可执行的修复命令
- 配置修改建议
- 历史成功修复案例

推荐位置：
- 告警详情页修复建议区
- 一键执行面板
```

## 2. 推荐算法

### 2.1 算法选型

| 推荐类型 | 算法 | 说明 |
|---------|------|------|
| 告警关联 | 向量相似度 + 规则 | 基于告警特征向量计算相似度 |
| 根因推荐 | LLM推理 + RAG | 大模型推理+知识库增强 |
| 修复推荐 | 案例匹配 + LLM生成 | 历史案例匹配+AI生成 |
| 知识推荐 | 向量检索 + 排序 | 知识库向量检索 |
| 功能推荐 | 协同过滤 | 基于用户行为相似度 |

### 2.2 推荐流程

```

└─────────────┘
```

## 3. 推荐效果评估

### 3.1 评估指标

| 指标 | 定义 | 目标 | 当前 |
|------|------|------|------|
| 点击率（CTR） | 推荐点击/展示次数 | >15% | 18% |
| 采纳率 | 推荐采纳/展示次数 | >10% | 12% |
| 准确率 | 正确推荐/总推荐 | >70% | 75% |
| 覆盖率 | 被推荐物品/总物品 | >60% | 65% |

### 3.2 A/B测试记录

| 测试ID | 测试内容 | 对照组 | 实验组 | 结果 |
|--------|---------|--------|--------|------|
| AB-01 | 根因推荐展示方式 | 列表展示 | 卡片展示 | 卡片+8% CTR |
| AB-02 | 推荐数量 | 3条 | 5条 | 3条+5%采纳率 |
| AB-03 | 置信度展示 | 不展示 | 展示置信度 | 展示+12%信任度 |

## 8. 推荐内容规范

### 8.1 推荐文案规范

| 规范项 | 要求 | 示例 |
|-------|------|------|
| 简洁明了 | <20字 | "推荐修复：重启服务" |
| 突出价值 | 说明收益 | "预计恢复时间：5分钟" |
| 置信度标注 | 标注可信度 | "置信度：85%" |

### 8.2 推荐展示规范

| 规范项 | 要求 |
|-------|------|
| 数量限制 | 每场景不超过5条 |
| 排序规则 | 按置信度/相关性排序 |
| 展示位置 | 不干扰主要操作 |
| 可关闭 | 用户可关闭推荐 |

## 9. 推荐伦理考量

### 9.1 公平性

- 避免推荐偏见
- 保证推荐多样性
- 不歧视特定用户群

### 9.2 透明性

- 推荐原因可解释
- 用户可查看推荐逻辑
- 提供反馈渠道

### 9.3 隐私保护

- 用户数据加密存储
- 推荐数据脱敏处理
- 符合数据安全法规
